NEWER OO FEATURES IN JAVASCRIPT

Getters
    allow you to retrieve the value of an object's property

Setters
    allow you to set the value of an object's property

Public Fields
    allow access within the class but also from
    outside the class or individual instances

Private Fields
    provide a way to maintain encapsulation and not
    allow external access

Static Initialization Blocks
    block of code that runs once when your class is loaded
